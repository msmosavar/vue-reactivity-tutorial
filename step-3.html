State چیچیه؟!

در واقع کلمه State در اکثر مواقع یه عبارت فانتزی برای "variables" یا «متغیر» هست.

وقتی به state یه برنامه اشاره می‌کنیم، داریم به یکی از مقادیری که داخل اون می‌تونیم تغییر بدیم اشاره می‌کنیم که در واقع هر مقداری که static نیست رو شامل میشه . توی جاوااسکریپت state ها این موارد هستن:

  • متغیرهای تعریف شده با let و const
  • خواص (properties) داخل object ها
  • مقادیر داخل آرایه‌ها یا سایر collection ها مثل map ها و set ها

این‌ها رو به عنوان root state نام‌گذاری می‌کنیم؛ یعنی مقادیر واقعی و مشخص (concrete). در مقابل derived state یا state مشتق‌شده داریم که با ترکیب کردن root state ها ایجاد میشن: 

        
          let a = 1;
          let b = 2;
  
          function aPlusB() {
            return a + b;
          }
