برنامه‌نویسی Declarative چیچیه؟!

پارادایم‌های برنامه نویسی مختلفی مثل imperative و declarative و functional و ... وجود دارن:

  • توی برنامه‌نویسی declarative برنامه‌نویس این که چه چیزی (what) باید رخ بده رو توضیح میده، بدون نگرانی در مورد این که چگونه (how) اون کار انجام میشه.
  • توی برنامه‌نویسی imperative برنامه‌نویس مراحل دقیق این که چگونه (how) چیزی باید رخ بده رو توضیح میده. 
        
          let a = 1;
          let b = 2;

          let aPlusB = a + b;

          // update `a`
          a = 4;
          aPlusB = a + b;

          // update `b`
          b = 7;
          aPlusB = a + b;
        
      
        
          let a = 1;
          let b = 2;

          function aPlusB() {
            return a + b;
          }
